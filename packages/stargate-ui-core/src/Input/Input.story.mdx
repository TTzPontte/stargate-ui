### States And Messages For Form Validation

At the moment, **Input** only supports messages for error statement and 1 per time.

<Canvas>
  <Input
    ref={ref}
    name="input8"
    label="Input With Error"
    value="Wrong value typed"
    error
    errorMessage="Mussum Ipsum, cacilds vidis litro abertis."
  />
</Canvas>

Stargate UI doesn't provide form validation control so your project needs to have it.

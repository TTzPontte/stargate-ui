## Controlled accordion

Accordion allows controlling opening individually.

<Canvas>
  <Story name="Controlled accordion">
    {
      (props) => {
        const [controller, setController] = React.useState({ opened: false });
        const onClick = (event, [controller, opened]) => {
          setController({ name: controller, opened: !opened });
        };
        return (
          <React.Fragment>
            <Accordion
              controller="accordion-1"
              opened={controller.name === 'accordion-1' && controller.opened}
              onClick={onClick}
            >
              <Accordion.Summary>
                #1 Accordion summary
              </Accordion.Summary>
              <Accordion.Details>
                <Typography>
                  Lorem, ipsum dolor sit amet consectetur adipisicing elit. Asperiores voluptates laborum assumenda excepturi cupiditate, dicta repellat odio minima expedita unde magni, omnis quaerat animi exercitationem ratione. Velit iure accusantium maxime?
                </Typography>
              </Accordion.Details>
            </Accordion>
            <Accordion
              controller="accordion-2"
              opened={controller.name === 'accordion-2' && controller.opened}
              onClick={onClick}
            >
              <Accordion.Summary>
                #2 Accordion summary
              </Accordion.Summary>

                  Lorem, ipsum dolor sit amet consectetur adipisicing elit. Asperiores voluptates laborum assumenda excepturi cupiditate, dicta repellat odio minima expedita unde magni, omnis quaerat animi exercitationem ratione. Velit iure accusantium maxime?

              </Accordion.Details>
            </Accordion>
